## 3. 레이블 데이터 명세 (Label Schema)

### 3.1. 학습/검증용 JSON 구조 (AI Hub)

AI Hub에서 제공하는 원본 메타데이터를 포함한 상세 JSON 구조입니다.

  --------------------------------------------------------------------------------------------------
  필드명 (Key)              데이터 타입              설명                  예시
  ------------------------- ------------------------ --------------------- -------------------------
  group_no                  Integer                  그룹 고유 번호        601031026000

  no                        Integer                  데이터 고유 번호      601031026053

  img_no                    Integer                  이미지 고유 번호      53

  catecode                  String                   카테고리 코드         `"060103"`

  cate1                     String                   (학습 사용) 과일      `"사과"`
                                                     대분류 (종류)         

  cate2                     String                   과일 중분류 (품종)    `"부사"`

  cate3                     String                   (학습 사용) 과일      `"특"`
                                                     소분류 (등급)         

  width                     String                   과일 실제 너비 (cm)   `"8.8"`

  height                    String                   과일 실제 높이 (cm)   `"8.1"`

  weight                    String                   과일 실제 무게 (g)    `"334"`

  copyright                 String                   저작권 정보           `""`

  repo                      String                   원본 저장소 경로      `"사과/부사/특/26"`

  identifier                String                   원본 이미지 파일명    `"53.png"`

  format                    String                   원본 이미지 포맷      `"png"`

  img_height                Integer                  이미지 픽셀 높이      1000

  img_width                 Integer                  이미지 픽셀 너비      1000

  date                      String                   촬영 일시             `"2020-11-09 01:55:18"`

  resolution                Integer                  해상도 (width ×       1000000
                                                     height)               

  bit                       String                   이미지 비트 수        `"24"`

  ...                       ...                      *(f_stop, iso, gps 등 ...
                                                     기타 메타데이터)*     

  truncated                 String                   객체 잘림 여부        `"on"`

  angle_direction           String                   촬영 각도 (방향)      `"top"`

  ...                       ...                      *(verticality_angle   ...
                                                     등)*                  

  **bndbox**                Object                   (학습 사용) 바운딩    `{"xmin": 0, ...}`
                                                     박스 객체             

  bndbox.xmin               Integer                  BBox 좌측 상단 X좌표  0
                                                     (pixel)               

  bndbox.ymin               Integer                  BBox 좌측 상단 Y좌표  0
                                                     (pixel)               

  bndbox.xmax               Integer                  BBox 우측 하단 X좌표  1000
                                                     (pixel)               

  bndbox.ymax               Integer                  BBox 우측 하단 Y좌표  1000
                                                     (pixel)               
  --------------------------------------------------------------------------------------------------
